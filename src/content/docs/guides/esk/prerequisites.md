## Read this first

- Unlocked bootloader.
- ADB and Fastboot installed.
- Backup your stock boot image so you can restore it if it fails to boot.

## Platform Tools

If you do not already have ADB and Fastboot, get Android Platform Tools here:

- [Platform SDK Download](https://developer.android.com/tools/releases/platform-tools#downloads)
